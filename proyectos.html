Lista de tareas con JavaScript

Lista de tareas es una aplicación web sencilla creada con JavaScript y bootstrap con la que se puede hacer seguimiento de tareas diarias: añadir tareas, tachar tareas completadas, deshacer el tachado y eliminar tareas. Las tareas introducidas se guardan en el localStorage, por lo que la información se mantendrá en la página aunque cerremos el explorador.
